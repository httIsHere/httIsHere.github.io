234.palindrome-linked-list

Author: httishere Date: 2021/10/13  16:31:14 Category: LeeCode

描述

Given the head of a singly linked list, return true if it is a palindrome.

判断一个链表是否为回文链表。

`题解

1. 暴力解法

将链表回文判断转化成数组回文判断。

2. 快慢指针

翻转链表前半段，并将链表前半段和后半段进行比较。
