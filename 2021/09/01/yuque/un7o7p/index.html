JS жү§иЎҢзІ’еәҰ Realm

  • JS Context => RealmпјҲзІ’еәҰжҜ”е®Ҹд»»еҠЎжӣҙеӨ§пјү
  • е®Ҹд»»еҠЎ
  • еҫ®д»»еҠЎ
  • еҮҪж•°и°ғз”Ё
  • иҜӯеҸҘ/еЈ°жҳҺ
  • иЎЁиҫҫејҸ
  • зӣҙжҺҘйҮҸ/еҸҳйҮҸ/thisвҖҰ

еҰӮжһңдҪҝз”Ё GLOBAL еҸҳйҮҸеҜ№иұЎжҳҜдёҚйңҖиҰҒз”ЁеҲ° RealmгҖӮ

еңЁ JS еҶ…пјҢеҮҪж•°иЎЁиҫҫејҸе’ҢеҜ№иұЎзӣҙжҺҘйҮҸеқҮдјҡеҲӣе»әеҜ№иұЎпјҢдҪҝз”Ё.еҒҡйҡҗејҸиҪ¬жҚўд№ҹдјҡеҲӣе»әеҜ№иұЎпјҢиҝҷдәӣеҜ№иұЎйғҪжҳҜжңүеҺҹеһӢзҡ„пјҢеҰӮжһңжІЎжңү Realm жҲ‘д»¬е°ұдёҚзҹҘйҒ“е®ғд»¬зҡ„еҺҹеһӢжҳҜд»Җд№ҲдәҶгҖӮ

иҺ·еҸ–жүҖжңү GLOBAL еҜ№иұЎ

е®Ҹд»»еҠЎ&еҫ®д»»еҠЎ

дёӨдёӘи§’иүІпјҡе®ҝдё»зҺҜеўғе’Ң JS еј•ж“ҺгҖӮ

  • е®Ҹд»»еҠЎпјҡе®ҝдё»еҸ‘иө·зҡ„д»»еҠЎпјӣ
  • еҫ®д»»еҠЎпјҡJS еј•ж“ҺеҸ‘иө·зҡ„д»»еҠЎпјӣ

JS жҳҜеҚ•зәҝзЁӢжү§иЎҢзҡ„гҖӮ

е®Ҹд»»еҠЎеҶ…йғЁдјҡжңүдёҖдёӘеҫ®д»»еҠЎеҲ—иЎЁпјҢеңЁжү§иЎҢдёӢдёҖдёӘе®Ҹд»»еҠЎд№ӢеүҚпјҢдјҡжҠҠдёҠдёҖдёӘе®Ҹд»»еҠЎеҶ…зҡ„еҫ®д»»еҠЎе…ЁйғЁе®ҢжҲҗгҖӮ

еҲҶжһҗејӮжӯҘжү§вҫҸзҡ„йЎәеәҸпјҡ

  • вҫёе…ҲжҲ‘д»¬еҲҶжһҗжңүеӨҡе°‘дёӘе®Ҹд»»еҠЎпјӣ
  • еңЁжҜҸдёӘе®Ҹд»»еҠЎдёӯпјҢеҲҶжһҗжңүеӨҡе°‘дёӘеҫ®д»»еҠЎпјӣ
  • ж №жҚ®и°ғвҪӨж¬ЎеәҸпјҢзЎ®е®ҡе®Ҹд»»еҠЎдёӯзҡ„еҫ®д»»еҠЎжү§вҫҸж¬ЎеәҸпјӣ
  • ж №жҚ®е®Ҹд»»еҠЎзҡ„и§ҰеҸ‘и§„еҲҷе’Ңи°ғвҪӨж¬ЎеәҸпјҢзЎ®е®ҡе®Ҹд»»еҠЎзҡ„жү§вҫҸж¬ЎеәҸпјӣ
  • зЎ®е®ҡж•ҙдёӘйЎәеәҸгҖӮ

жҖқиҖғдёӢйқўд»Јз Ғзҡ„жү§иЎҢйЎәеәҸпјҡ

дјҡе…ҲеңЁдёҖдёӘеҫ®д»»еҠЎеҶ…жү§иЎҢе®ҢжүҖжңүзҡ„еҗҢжӯҘд»Јз ҒгҖӮ

е®Ҹд»»еҠЎ 1:

  • еҫ®д»»еҠЎ 1пјҡ0пјҢ4пјҢ5пјҢ-2пјҲеҗҢжӯҘд»Јз Ғпјү
    • е…Ҙйҳҹпјҡ1пјҢ-1
  • еҫ®д»»еҠЎ 2пјҡ1
    • е…Ҙйҳҹпјҡ1.5
  • еҫ®д»»еҠЎ 3пјҡ-1
  • еҫ®д»»еҠЎ 4пјҡ1.5

е®Ҹд»»еҠЎ 2:

  • еҫ®д»»еҠЎ 1пјҡ2
  • еҫ®д»»еҠЎ 2пјҡ3

жү§иЎҢдёҠдёӢж–Үж ҲпјҲExecution Context StackпјүгҖӮ
